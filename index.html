Rules for Naming JS Variables

Variable name can only contain letters, numbers,underscores(_),and doller signs ($). For example: $my_1stVariable

Variables must begin with a letter, underscore (_), or dollar sign ($). For example: $name, _name, or name.

Variable names are case-sensitive.

Variable names should not be JavaScript reserved keywords.
